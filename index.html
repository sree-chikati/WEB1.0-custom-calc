GDP Multiplier Formulas

GDP MULTIPLIER FORMULA DISPLAY

MPC =

1 (Make sure this is 1) - MPS

MPC =

Change in Consumption / Change in Income

MPS =

1 (Make sure this is 1) - MPC

MPS =

Change in Savings / Change in Income

Tax Multiplier =

MPC / MPS

Inflation Formulas

INFLATION FORMULAS DISPLAY

Inflation =

Nominal % change - Real % change

Real % Change =

Nominal % change - Inflation

CPI =

New Market Basket Value / Base Market Basket Value x 100

Deflator =

Nominal Value / Real Value x 100

Inflation Rate =

(New Index - Old Index ) / Old Index x 100

Real Value =

Nominal Value / Index x 100
